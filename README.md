This is a polls app made for the course Cyber Security Base Project 2025 at the University of Helsinki. 
The app contains five common security flaws from the [OWASP top ten list 2021](https://owasp.org/Top10/2021/).

## How to install and use

> [!NOTE]
> In order to use the app, python3 and Django should be installed.

Clone the project to your own computer:
```
$ git clone https://github.com/ronjass/CyberSecurityBase20205.git
```
Navigate to the root of the project directory and apply migrations:
```
$ python3 manage.py migrate
```
Start the Django development:
```
$ python3 manage.py runserver
```
